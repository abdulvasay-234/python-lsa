### **Python Assignment: Restaurant Billing System**

Write a Python program to simulate a simple restaurant billing system.

**Requirements:**

1. Create a menu of food items and their corresponding prices using **lists**.
2. Ask the customer to enter the names of **any two food items** they have ordered.
3. Calculate the total bill based on the prices of the selected items.
4. Calculate **18% GST** on the total bill.
5. Display:

   * The selected items
   * Total amount
   * GST (18%)
   * Final bill (including GST)

**Constraints:**

* Use only:

  * Variables
  * Lists
  * `input()` and `print()`
  * Arithmetic operators (`+`, `-`, `*`, `/`, `%`)
  * Type casting (if required)
  * List methods such as `index()`
* Do **not** use:

  * `if`, `elif`, `else`
  * Loops (`for`, `while`)
  * Functions
  * Dictionaries

**Note:** If the user enters an item that is not available in the menu, Python's default error (`ValueError`) is acceptable.
